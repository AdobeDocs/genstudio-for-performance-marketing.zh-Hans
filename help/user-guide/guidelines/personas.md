# GenStudio for Performance Marketing [!DNL Personas]

[!DNL Personas]提供客户细分的真实描述，包括客户的信息、兴趣、痛点、偏好和行为特征。

GenStudio for Performance Marketing利用[准则的强大功能](overview.md)和Adobe的创作AI技术为您的目标客户策划个性化营销内容&#x200B;。

**要查看或修改角色**：

1. 在左侧导航区域中，单击&#x200B;**[!UICONTROL 角色]**。
1. 要[查看个人详细信息或修改角色](add-guidelines.md#manage-personas)，请单击&#x200B;_[!DNL Personas]_&#x200B;操作菜单，然后选择&#x200B;**打开**。
1. 通过单击准则类别文本字段来修订角色准则。

请参阅[添加准则](add-guidelines.md)，了解如何在GenStudio for Performance Marketing中添加或修改角色。

## [!DNL Personas]指南

个人角色准则为GenStudio for Performance Marketing提供了您[!DNL Personas]的清晰图像。

![角色准则](/help/assets/personas.png){width="600" zoomable="yes"}

这些指南详细介绍了每个指南类别和相关示例。 这些示例是构成所生成内容输出的输入示例。 使用它们通知您自己的[!DNL Personas]指南。

下表显示了[!DNL Personas]指南的每个类别和相关示例。

| 准则类别 | 准则示例 | 最佳做法 |
| ------------------| :---------- |-------------|
| 角色名称 | 代表量身定制营销材料的特定受众区段 | 添加简洁明了的角色名称。<br><br>_示例_：“零售业的CMO”、“未来的护理学生”、“新的储蓄帐户客户” |
| 描述 | 定义关键特征、需求和行为的简短摘要 | 使用简洁的语言清晰地描述角色和独特的受众区段（1-2个句子）。<br><br>关注此受众区段的驱动因素（需求、核心特征等）<br><br>保持描述简洁且不包含深层人口统计数据或额外信息。 |
| 消息首选项 | 明确总结与特定角色产生共鸣的首选通信样式 | 定义角色独特的消息传递要求和动机，关注角色接收和与之交互的偏好设置。<br><br>在列表中提供特定的消息传递偏好设置（每个列表项1-2个句子）。<br><br>将列表限制为总共10个句子或更少，以确保突出显示最重要的信息。<br><br>_示例_： “消息传递偏好设置对于精通技术的专业人员：<li>更喜欢简洁、没有术语的沟通。</li><li>积极响应数据驱动型洞察和切实可行的建议。</li><li>首选可视化（如信息图和图表）以快速掌握复杂信息。</li><li>以诚实、直截了当的方式重视透明度。</li>&quot; |
