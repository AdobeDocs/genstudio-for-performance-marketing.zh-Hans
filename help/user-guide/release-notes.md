# GenStudio for Performance Marketing发行说明

此发行信息提供了对GenStudio for Performance Marketing应用程序的最新更新。

## 2026.04 {#latest}

### Meta广告帐户按操作类型划分的CPA

对于[!DNL Insights]中的Meta广告营销活动，您现在可以使用与优化营销活动的方式匹配的转化操作来分析每次操作的成本(CPA)，而不是只使用单个默认量度。

从可用的Meta操作类型中进行选择，例如：

* 购买
* 添加到购物车的事件
* 销售线索
* 视频观看次数
* 自定义转化

并在广告、Assets （媒体）和属性[!DNL Insights]表中以一致的方式审核CPA。

### 创建工作流中的RTCDP受众

[Adobe Real-Time Customer Data Platform (RTCDP)受众](https://experienceleague.adobe.com/en/docs/genstudio-for-performance-marketing/user-guide/guidelines/add-guidelines#add-audiences)在[!DNL Create]工作流中可用，因此您可以在受众定义中与现有准则一起构建生成的体验。

### 在[!DNL Insights]中没有显示渠道

Innovid现在在[!DNL Insights]中显示为显示渠道，因此您可以在一个位置查看Meta、LinkedIn、TikTok以及其他连接的源旁边的营销活动和广告效果，并且每天都会刷新指标。

使用Innovid评估展示次数、点击量和点进率。

### 通过单个视频资产激活LinkedIn

现在，您可以按照与静态LinkedIn广告相同的端到端激活流程，从GenStudio for Performance Marketing将使用单个视频资源[&#128279;](/help/user-guide/activation/activate-linkedin-ad.md)的LinkedIn广告体验激活到LinkedIn促销活动管理器。

### 针对付费媒体和电子邮件体验的现成翻译

现在使用[创建主页](/help/user-guide/create/overview.md)上的&#x200B;**[!UICONTROL 翻译和本地化]**&#x200B;卡片，或者在[管理资产和体验](/help/user-guide/content/manage-assets.md)中从已批准的付费媒体体验或电子邮件体验的“操作”菜单中选择&#x200B;**[!UICONTROL 翻译]**&#x200B;以翻译内容

对于付费媒体，请将内容翻译为&#x200B;**40+**&#x200B;种语言。

对于电子邮件，请在单个流程中将内容翻译为&#x200B;**超过10**&#x200B;种支持的语言。

### 模板资产的第三方DAM连接

组织现在可以使用注册的加载项来&#x200B;**连接自己的数字资产管理(DAM)**&#x200B;存储库，以便作者可以在模板参数流中浏览并从该系统选择媒体。

请参阅[查找DAM扩展](/help/extensibility/deploy-app.md#find-dam-extensions)和[开始构建可扩展的应用程序](/help/extensibility/setup.md)，以了解如何构建和分发加载项。

### 用于集成的Experience Selector微前端

**Experience Selector**&#x200B;将GenStudio引入合作伙伴和自定义应用程序中。 嵌入微前端(MFE)，以在合作伙伴或自定义应用程序中提供符合您集成要求的、具有一致UI和登录的浏览、筛选和选择功能。

在[GenStudio Experience Selector MFE](/help/extensibility/experience-selector.md)中采用独立捆绑包、配置选项和嵌入模式。 您的团队会提供选取器体验，而无需从头开始重新构建选择UX。

### Firefly Image Model 4上的自定义模型

在[!DNL Create]中[生成图像](/help/user-guide/create/generate-assets.md)。 **[!UICONTROL 模型]列表现在包含在Adobe Firefly**&#x200B;图像模型4上针对&#x200B;**品牌**&#x200B;和&#x200B;**文本到图像**&#x200B;工作进行培训的自定义模型，这些模型反映的是图像模型4的培训情况，而不只是基于图像模型3的旧自定义模型。

您仍可停留在相同的[!DNL Create]流程中，而较新的Firefly基础支持使用经过图像模型4训练的模型来进行品牌安全生成。 参数和提示的行为与其他模型选取类似 — 没有可直立的并行工作流。

### Horizon画布上的快速模板（Meta、LinkedIn和Display）

**Horizon画布** + Adobe Express模板现在为[!DNL Create]中的&#x200B;**Meta**、**LinkedIn**&#x200B;和&#x200B;**显示**&#x200B;的&#x200B;**正式提供**。 多尺寸布局、移动就绪型Express设计（如果支持）、图层锁定和WYSIWYG编辑可替代这些渠道的旧HTML模板瓶颈，因此团队可以更快地提供品牌创意。

阅读[使用GenStudio for Performance Marketing中的Adobe Express模板](/help/user-guide/templates/express-templates.md)以了解支持的格式、入门和自定义模板以及画布功能。 本主题说明了扩展Express布局时提供的特定于渠道的支持。

### PDF导出以供审阅和存档

[!BADGE Beta]{type=Informative tooltip="此功能当前位于Beta中，因此某些功能可能会受到限制或发生更改。"}

您可以[从[!DNL Create]画布上的导出草稿和[!DNL Content]中的已批准体验中将体验导出到PDF](/help/user-guide/content/manage-assets.md#export-experiences)，从而生成一个不可编辑的文档用于利益相关者审核、批准记录和存档 — 尤其是当团队需要固定快照用于合规性或受管控的审核工作流时。

* 用户可以配置视频广告创意详细信息，预览视频的显示方式，并向Meta平台激活视频广告。

### 适用于Meta促销活动的按操作类型划分的CPA

* 针对Meta广告促销活动，通过灵活的转化跟踪[&#128279;](/help/user-guide/insights/ads.md)分析每操作成本(CPA)。
* 从多种操作类型（如购买、添加到购物车的事件、商机、视频查看或自定义转化）中进行选择（而不是单个固定量度），以在[!DNL Insights]中跨促销活动、广告、资源和属性计算和查看CPA。

* 添加了SDK扩展，该扩展允许客户连接到其自己的DAM，而不是在内容生成期间使用AEM Assets作为资产的默认值。

* [!DNL Experience Translations]流已随付费媒体和电子邮件的新翻译功能一起优化。

这当前仅适用于英文广告。 有关详细信息，请参阅[文本功能](/help/user-guide/insights/text-features.md)。

* 用于保存激活草稿的新功能。
* 用于重试失败的激活的新功能。

### 在多个文本字段中使用相同的角色

现在支持具有相同角色的多个文本字段（例如，“body”、“cta”、“on image text”等） 适用于复杂的客户模板。

在[有关模板代码编辑器的指南](/help/user-guide/templates/code-editor.md)中浏览详细信息。

### 兼容的资产过滤器

[!DNL Insights]模块中的新筛选器会自动从广告预览中隐藏[不受支持的图像和视频资源](/help/user-guide/insights/ads.md#ad-formats)，从而消除视觉混乱和磁贴损坏。 此增强功能可确保用户仅看到实际可用且准备使用的媒体，从而创造更干净和更可靠的体验。 该过滤器与现有兼容性广告过滤器配合使用。

### 功能

主要产品功能包括：

**[!DNL Create]**&#x200B;引入了Canvas，它提供了一种结构化的提示体验，使内容编辑者能够快速生成内容和变体。 系统管理员根据组织品牌准则对产品进行培训。 [!DNL Create]确保所有AI生成的内容都与您的品牌准则（品牌策略、客户角色和产品描述）保持一致，并简化产生高影响力、品牌一致的营销内容。

**[!DNL Content]**&#x200B;存储策划的、符合品牌标准的已批准资产和体验。 GenStudio for Performance Marketing用户可以轻松查找、编辑、重新调整用途和共享经过批准的资源，从而无需为每个活动从头开始重新创建内容。

**[!DNL Reviews and Approvals]**&#x200B;建立了一个框架，供利益相关者在保存到&#x200B;**[!DNL Content]**&#x200B;或导出之前审核和批准生成的变体。

**[!DNL Campaigns]**&#x200B;组织和管理营销活动，确保优化执行和跟踪。 协作者可以可视化、规划和跟踪活动，从而有效地管理多个计划并确保及时交付。

**[!DNL Insights]**&#x200B;提供内容性能的实时评估，帮助营销人员优化其策略并做出数据驱动型决策。

GenStudio for Performance Marketing与其他Adobe Experience Cloud产品集成，包括Adobe Express和Adobe AEM Assets。
