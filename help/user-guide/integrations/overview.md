# Adobe GenStudio for Performance Marketing集成

GenStudio for Performance Marketing提供了与其他功能强大的设计产品的集成，以便为依赖这些产品的营销人员提供无缝体验。 您将在此处找到有关可用集成及其使用方式的信息。

浏览GenStudio for Performance Marketing提供的插件以及其他Adobe产品及其他内容，以简化您的设计工作流。

## Figma插件

GenStudio for Performance Marketing Figma插件为Figma应用程序添加了一个新面板，通过该面板，用户可以执行GenStudio for Performance Marketing操作，如生成品牌内内容。

在[Figma插件](figma-plugin.md)页面中了解有关Figma插件的更多信息。

## Photoshop插件

GenStudio for Performance Marketing Photoshop插件在Adobe Photoshop中新增了一个面板，通过该面板，用户可以执行GenStudio for Performance Marketing操作，如生成品牌内内容。

在[Photoshop插件](photoshop-plugin.md)页面中了解有关Photoshop插件的更多信息。
