# 创建连接的电视体验

在[!DNL GenStudio for Performance Marketing]中使用[[!DNL Create]](/help/user-guide/create/overview.md)在一个位置生成连接电视(CTV)广告 — 从简短和共享的准则到生成、基于场景的细化、批准和发布者就绪的导出。 以下工作流完全在[!DNL GenStudio for Performance Marketing]中运行；没有单独的CTV应用或发布者嵌入。

## 先决条件

在创建CTV广告之前，请确认以下各项：

* 访问[!DNL GenStudio for Performance Marketing]。
* **[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**&#x200B;在[!DNL GenStudio for Performance Marketing]中配置为共享对象。 请参阅[准则概述](/help/user-guide/guidelines/overview.md)以了解这些对象如何通知生成。
* 推荐使用Campaign资产（视频剪辑、图像、徽标、音乐），但不一定需要，创作AI可以在资产缺失或不完整时填补空白。

## 创建新的CTV广告

此工作流中的所有操作均发生在[!DNL GenStudio for Performance Marketing]内。

![已连接电视创建卡](./ctv-tile.png){width="50%"}
**导航到CTV创建**：

1. 登录到[!DNL GenStudio for Performance Marketing]。
1. 从主页或创建表面，转到&#x200B;**[!UICONTROL 创建]**。
1. 使用CTV创建卡选择&#x200B;**CTV**。
1. 单击&#x200B;**[!UICONTROL 创建CTV广告]**。

将打开一个简化的CTV创建体验。 您无需首先选择广告类型。

## 配置摘要

简介和输入可驱动广告的生成。 这是您为广告生成过程提供上下文和限制的机会。

![配置摘要](./ctv-brief.png){width=80%&quot; align=&quot;center&quot;}

**配置简报**：

1. 从现有共享对象中选择&#x200B;**[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**。
1. 通过直接输入或上传&#x200B;**创意摘要**&#x200B;来添加它。 包括营销活动目标、关键消息和任何限制。
1. 将&#x200B;**广告持续时间**&#x200B;设置为15秒或30秒。
1. 可以选择添加&#x200B;**资源**。 上传视频剪辑、图像、徽标、音乐、画外音或简介/简介卡（拖放或文件选择），或者从[!DNL Content]存储库中选择资源。
1. 单击&#x200B;**[!UICONTROL 生成]**&#x200B;按钮。

如果资源缺失或不完整，[!DNL GenStudio for Performance Marketing]可以使用AI生成缺失的场景、音乐或画外音。 您提供的Assets始终优先于生成的材料。

[!DNL GenStudio for Performance Marketing]自动：

* 解释摘要以及来自&#x200B;**[!DNL Brands]**、**[!DNL Products]**&#x200B;和&#x200B;**[!DNL Personas]**&#x200B;的上下文。
* 组合了完整的CTV广告结构。
* 根据需要创建场景、文本叠加、音乐和画外音。
* 应用符合CTV标准的持续时间和格式。

结果是一个完全格式化、可预览的CTV广告 — 而不是简单的草稿时间线。

## 编辑和优化广告

使用基于场景的编辑器来优化广告，而不重新生成所有内容。

单击场景带中的场景以打开它进行编辑。 您可以执行的编辑包括：

* 用AI替换或重新生成单个场景。
* 编辑场景提示以创建变体。
* 重新排序或修剪场景。
* 编辑文本叠加。
* 交换、静音或替换音乐和画外音。
* 调整场景之间的过渡。

编辑属于范围，因此您可以一次重新生成一个场景，以加快迭代和创意刷新。

>[!NOTE]
>
>编辑器不支持更改视频剪辑中的对象&#x200B;**（例如，删除项目、更改产品颜色或更改人员的显示方式）。

## 审阅并批准

使用集成的审批工作流提交广告以供品牌审阅。 品牌和利益相关者审阅者检查消息传送、视觉效果和品牌合规性。 审批者验证广告；他们不应代替营销人员执行视频编辑。

## 导出

批准后，您可以：

* 以出版商就绪且符合要求的格式导出最终确定的CTV广告。
* 将广告保存回[!DNL Content]。
* 在下游CTV购买和贩运工作流中使用它。

Creative旨在做好激活准备，无需重新编码或返工。
